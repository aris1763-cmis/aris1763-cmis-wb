Lists and Links

Assigment

This assigment is a simple tutorial on how to create a new webpage on the production server. Links leading to other helpful websites are also included to help better understand each step.

  • Create a new folder called "listsandlinks" in your repository folder.
  • Inside listsandlinks create a file called "index.html" and a file called "main.css"
  • In index.html create an ordered list of step by step instructions to create a new webpage on the production server.
  • Also create an unordered list of 5 or more links to websites with useful webdesign related information.
  • In the nav section, create an unordered list of links to with a link back to your main index page.
  • In your main index.html file in public_html, add an unordered list of links to the nav section with a link to "listsandlinks"
  • Add an article with a heading and a paragraph description of this assignment. Also add an unordered list that contains this list of requirements.
  • Turn in your work.

Creating a New Webpage on the Production Server

  1. Create a .html file and edit the files.
  2. Validate the file.
  3. Open python and cd into your folder containing the .html file
  4. In python, start sftp with the command sftp username@cmis-hsteacher.
  5. cd into public_html.
  6. Use the command put (filename) to upload the .html file from your computer to the production server.
